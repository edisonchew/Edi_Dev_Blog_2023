Week 2 - Semantics & CSS

Semantics

  • Relates to syntax
  • The meaning behind words, phrases. (In our case it is the HTML)

What I have learnt:

Tags:

  • Header (header) --> Used for navigation or introductory content (Can be used at the top of the page content to encapsulate navigation, logos
  • Article (article) --> Commonly used for news content, blog posts or independently content piece
  • Aside (aside) --> For content that is indirectly related to your main content (Commonly seen in sidebars or call out boxes)
  • Style (style) --> Defines style information for a document
  • Link (link) --> Defines the relationship between a document and an external resource (most used to link to style sheets)

ID Attribute - TARGETING AN INDIVIDUAL:

  • #header{} selects any element with the id "header", (p class = "header" /p)
  • Unique to that element
  • Hash symbol
  • The "#" is how you tell CSS "this is an id".

Class Attribute - TARGETING A GROUP:

  • .warning { color: red; } Selects any element with the class name "warning", e.g. (p class = "warning" /p)
  • Used to style an element grouping
  • Targets one or more element's class attribute
  • Elements can be of different type
  • An element can have only 1 id but multiple classes, so you can take advantage of that for greater flexibility

Fonts:

  • font-size --> It is a property specifies the size of a font (Can be specified as a fixed size in various units, a percentage, or as predefined keyword
    • font-size: 1.5em;
    • font-size: 12px;
    • font-size: 100%
    • font-size: larger;
  • "em' unit:
    • Lets you set the size of the text relative to the text around it.
    • Makes the page resize nicely proportion if the user changes their default font-size.
    • The default size is "1em" or 16px
    • e.g. p { font-size: 0.9em; }, strong { font-size: 1.5em; }
  • "px" unit:
    • Lets you size font in terms of pixels, which is the unit also used to size images and other elements
    • Easier to understand than em
    • Dont work as well when printing or resizing
    • e.g. h2 { font-size: 17px }
  • font-style --> Specifies the font style of the text, either "normal" by default or "italic"
  • font-weight --> Specifies the thickness of the font (Default = "normal"; Typical = "bold"; can be "bolder, "lighter", or number from 100 to 900)
